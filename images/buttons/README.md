# Button Icons

Store menu button icons in this folder (`images/buttons`).

Current icon style/source:
- [Icons8 Forma Regular Filled](https://icons8.com/icons/forma-regular-filled)
- You can download more icons from the same set to match the existing look.

Recommended format:
- `PNG` file (`.png`)
- `96x96` pixels
- Transparent background

Notes:
- Other icon sets are also possible, but this Icons8 set matches the current UI best.
- Keep high contrast so icons stay visible on active/inactive button colors.
- Use simple shapes; avoid tiny details that disappear on the touchscreen.
- Use short, lowercase filenames with dashes (example: `volume-up.png`).
