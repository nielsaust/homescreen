# Button Icon Guidelines

Store menu button icons in this folder (`images/buttons`).

Recommended format:
- `PNG` file (`.png`)
- `96x96` pixels
- Transparent background preferred

Practical notes:
- Keep high contrast so icons stay visible on active/inactive button colors.
- Use simple shapes; avoid tiny details that disappear on the touchscreen.
- Use short, lowercase filenames with dashes (example: `volume-up.png`).
